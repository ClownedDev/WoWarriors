# LegionCore
LegionCore 7.3.5
----------------
World of Warcraft Legion 7.3.5 server core, based on TrinityCore.




Requirements:
-------------
All requirements for Win64 included and preconfigured to cmake.

- World Of Warcraft 7.3.5 (26972)
- Visual Studio 2019 Win64
- Boost 1.64 Win64
- Cmake 3.16.4 Win64
- MySQL 5.6.47
- OpenSSL 1.1.1e Win64
